# Guide: Flags and capture groups

Flags change matching: `g` finds every match (not just the first), `i` is case-insensitive, `m` makes the anchors match per line, `s` lets the dot match newlines. Parentheses create capture groups — a pattern like user@domain with parentheses captures each part separately, which is how you extract structured data from text.
