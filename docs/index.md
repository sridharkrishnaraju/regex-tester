# Regex Tester

Test a regular expression against any string with live match highlighting and capture groups. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="regex-tester.js"></script>
<regex-tester></regex-tester>
```

## What it does

A regular expression is a pattern for matching text — finding, validating or extracting parts of a string. This tester uses JavaScript regex syntax, highlights every match live, and lists the capture groups (the sub-strings your parentheses extract).

## When you'd use it

Debugging a tricky pattern, checking what a regex actually matches, or building up a validation rule step by step.

See the [Usage & API guide](usage.md) for framework examples, or the [guide](guide.md) for tips.
