# Usage & API

It is a standard custom element, so it works with no wrapper in plain HTML, React, Vue, Svelte and Astro.

## Plain HTML

```html
<script src="regex-tester.js"></script>
<regex-tester></regex-tester>
```

## React

```jsx
import "@sgbp/regex-tester";
export default function Page() { return <regex-tester />; }
```

## Vue

```vue
<script setup>
import "@sgbp/regex-tester";
</script>

<template>
  <regex-tester />
</template>
```
