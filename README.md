# Regex Tester

Test a regular expression against any string with live match highlighting and capture groups. Zero dependencies — works in plain HTML, React, Vue, Svelte or Astro.

**▶ [Live demo](https://sgbp.tech/tools/regex-tester)**

```html
<script src="regex-tester.js"></script>
<regex-tester></regex-tester>
```

## What it does

A regular expression is a pattern for matching text — finding, validating or extracting parts of a string. This tester uses JavaScript regex syntax, highlights every match live, and lists the capture groups (the sub-strings your parentheses extract).

## Install

```bash
npm install @sgbp/regex-tester
```

or copy `regex-tester.js` into your project.
